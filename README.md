# LinkPro Analytics - Internship Project
Internship Project - Analytics dashboard and a simple recommendation system for the website LinkPro.

## Overview
Building an analytics dashboard for LinkPro to track:
- Link clicks and performance
- Traffic sources (Instagram, TikTok, etc.)
- User behavior patterns
- Performance recommendations

## Tech Stack
- **Backend:** Python + FastAPI
- **Database:** PostgreSQL
- **Analytics:** Pandas + NumPy
- **Frontend:** React (simple dashboard)
- **Data Exploration:** Jupyter Notebooks
